# Sito: Concetti fondamentali del Calcio

File principali creati:

- `index.html` — pagina principale con spiegazioni in italiano
- `styles.css` — stile semplice e responsivo
- `script.js` — piccoli miglioramenti UX (scroll morbido)

Contenuti aggiornati:

- sezione "Calcio femminile": criteri base, sicurezza e sviluppo giovanile
- sezione "Comportamento in squadra": valori e comportamenti pratici
- sezione "Benefici": vantaggi fisici, mentali e sociali

Per aprire il sito in locale, nella root del repository esegui:

```bash
python3 -m http.server 8000
# poi apri http://localhost:8000/index.html nel browser
```

Modifica i contenuti direttamente in `index.html` per aggiungere altri argomenti o esempi.
